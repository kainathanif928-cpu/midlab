⚠️ Vercel Deployment Note:

If you are seeing this page instead of your actual application, it means Vercel is serving the root folder instead of the Next.js app.

To fix this and make your actual app live:
  1. Go to your project settings on Vercel.
  2. Find the "Root Directory" setting.
  3. Click Edit and type AdFlow Pro.
  4. Save and Redeploy!
